🏥 Hospital Management System
A Java Swing-based desktop application for managing patients, doctors, and appointments. This project is designed to streamline hospital operations with an interactive UI and database-driven backend.

🚀 Features
📊 Dashboard – Real-time statistics of patients, doctors, and appointments
👨‍⚕️ Doctors Management – Add, edit, delete, and view doctors with specialization
🧑‍🤝‍🧑 Patients Management – Store and manage patient details (name, age, gender)
📅 Appointments – Schedule, update, and cancel patient-doctor appointments
🔄 Automatic Dashboard Updates – Changes reflect instantly across tabs
🎨 Modern UI – Styled with FlatLaf for a clean look

🛠️ Tech Stack
Frontend: Java Swing (GUI)
Backend: Java
Database: MySQL
Libraries: JDBC, FlatLaf (for UI)

📌 Future Enhancements
User authentication (Admin/Staff login)
Prescription and billing management
Export reports (PDF/Excel)
Cloud-hosted MySQL integration
